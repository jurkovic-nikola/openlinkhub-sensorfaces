# OpenLinkHub KDE System Monitor Sensor Face

### Install
- Extract org.openlinkhub.openlinkhub folder to `~/.local/share/ksysguard/sensorfaces/`
- If `~/.local/share/ksysguard/sensorfaces/` does not exist, create complete path chain, starting from `ksysguard` and `sensorfaces`
- Start KDE System Monitor application
- In sidebar, click on Menu icon and `+ Add New Page...`
- Name your page OpenLinkHub, pick any Icon and click on `+ Add`
- Click on newly created page `OpenLinkHub` and click on pencil `Edit Page` on the right side
- Click in the middle of Pie Chart and from `Display Style` select `OpenLinkHub`
- Tweak any settings if needed and click on `Save Changes`
